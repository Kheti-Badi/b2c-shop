**Note**: Make sure your Flutter environment is setup.

In the command terminal, run the following commands:

    $ git clone https://github.com/emrade/flutter-ui-fruit-store.git fruit_store
    $ cd fruit_store/
    $ flutter packages get
    $ flutter run
